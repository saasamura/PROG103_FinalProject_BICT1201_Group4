# Business Management System

## Project Overview

The Business Management System is a desktop application developed using Python, Tkinter, SQLite, Matplotlib, and ReportLab. The system was developed as a final project for the **Principles of Structured Programming (PROG103)** course.

The application helps organizations manage records efficiently through a graphical user interface (GUI), providing authentication, record management, search and filtering capabilities, data visualization, and PDF report generation.

---

## Features

### User Authentication

* Secure login page
* Username and password validation
* Error handling for invalid credentials

### Record Management

* Display records in a dashboard
* Add new records
* Store records using SQLite database
* View all records

### Search Functionality

Users can search records using:

* Record ID
* Full Name
* Status
* Contact Information

### Filter Functionality

Records can be filtered by:

* Gender

  * Male
  * Female
  * Other
* Status

  * Active
  * Inactive
  * Pending

### Data Visualization

The system provides:

* Bar Chart
* Pie Chart
* Line Graph

### PDF Reports

Generate:

* Weekly Reports
* Monthly Reports
* Yearly Reports
